安装教程

https://zhuanlan.zhihu.com/p/51224259
按照此教程：

  • 首先安装anconda

  • 然后创建新的环境 选python3.6

  • 在新的环境里打开终端安装tensorflow-gpu

  • 终端输入：conda install -c aaronzs tensorflow-gpu==1.12

  • 终端输入：conda install -c anaconda cudatoolkit

  • 终端输入：conda install -c anaconda cudnn

继续安装其他依赖的库

  • 终端输入： conda install seaborn
  • 终端输入： conda install matplotlib

下载sumo

  • sumo官网下载最新版即可

设置环境变量

SUMO_HOME=/usr/share/sumo
PYTHONPATH=$SUMO_HOME/tools:$PYTHONPATH

下载项目代码包

  • https://github.com/cts198859/deeprl_signal_control
  • 安装pycharm：百度搜索官网即可
  • pycharm下载较早版本
  • https://download.jetbrains.com/python/pycharm-community-anaconda-2020.1.5.exe?_ga=2.216294486.1780916785.1609925395-941539306.1603163253
  • pycharm破解包
  • 链接:https://pan.baidu.com/s/1HIHAqtE1a7ERGjnNaazkUg 密码:xwxv

运行：

  • pycharm 选择解释器

    选择创建的虚拟环境的python解释器
  • pycharm 编辑运行参数
  • 如果pycharm运行不行，就打开终端 cd到main.py文件下，然后在终端输入

python3 main.py –base-dir [base_dir]/[agent] train –config-dir [config_dir] –test-mode no_test
要先创建一个base_文件夹 agent在iqll ia2c ma2c里选 ，config在config文件夹里找对应的ini文件
